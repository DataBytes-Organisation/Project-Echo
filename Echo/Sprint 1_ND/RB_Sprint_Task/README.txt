
Project Contents

This repository includes the following components:

- Animal Sounds Tracker 
  An Excel spreadsheet used to track the collection and classification progress of animal sound samples across multiple species.

- CNN Classification Model  
  A convolutional neural network (CNN) implemented in PyTorch, designed to classify spectrogram images derived from animal audio recordings.
